my shell config for ``dash`` ``ash`` and ``bash``.

## Build

for ``bash``:

```shell
$ make bash
```

for ``ash``:

```shell
$ make ash
```

for ``dash``:

```shell
$ make dash
```

... and check generated file ``shell-config``.
